网络术语

June 8, 2021

Ethernet

以太网，是局域网最常见的组网方式。最简单的以太网就是多个主机通过一根电缆连接在一起。

复杂一些的以太网则通过双绞线、无线、集线器等设备连接在一起。以太网中报文碰撞现象的发生比较频繁，繁忙时可能使带宽下降至 30% 。

LAN

局域网（ Local Area Networks ）的缩写。

packet switching

分组交换，是多个用户共享同一个网络连接的解决方案。 TCP/IP 是一种实现分组交换的协议（栈）。

TCP

传输控制协议（ Transmission Control Protocol ）的缩写。

WAF

Web 应用防火墙（ Web Application Firewall ）。 WAF 部署在 Web Application 的前端，主要用于防御针对网络应用层的攻击，例如， SQL 注入，跨站脚本攻击，参数篡改、应用平台漏洞攻击、拒绝服务攻击等。
