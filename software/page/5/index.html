AWS Lambda

简介 术语 术语 描述 layer 一个层是一个 .zip 文件（不适用于镜像），使用层技术，可以使部署包尽可能地小，因为部署包依赖的库可以通过一个新的层提供 runtime 在 Lambda 运行环境中工作的一个编程语言相关的环境，负责转发调用事件，上下文信息，以及最终的响应 handler 函数代码中由 Lambda 调用并开始执行的方法 code signing 代码签名 execution role 授权 Lambda 函数去访问 AWS 服务和资源的 IAM 角色（ AWS Identity and Access Management (IAM) role ） unpublished version 未发布的版本，当函数更新时，会发生相应的变化 published versions 已发布的版本，是函数的一个快照，不会发生变化 aliases 函数别名，可以设置使之对应于一个版本 ARN Amazon Resource Name ， 例如，execution role ARN 的样式为 arn:aws:iam::your-account-id:role/execution_role 常用命令 AWS CLI 卸载 AWS CLI version 1
