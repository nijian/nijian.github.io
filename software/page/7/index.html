AWS Lambda

简介 AWS Lambda 是 AWS 提供的一种计算服务，服务内容包括： 提供高可用性计算基础设施 提供计算资源的自动管理任务，包括服务器和操作系统维护、容量配置和自动伸缩、代码监控和日志记录等 仅在需要时运行 Lambda Function ，并根据计算时间计费 Lambda Function 是部署在 Lambda 服务上的程序包。仅在需要时运行 Lambda Function ，并根据计算时间计费，这项服务内容可以极大地降低云平台的使用成本，事实上，这也是 Lambda 服务的核心价值。 使用 Lambda 服务会对编程思想带来一定的改变，例如， Lambda Function 应该尽可能做到无状态和原子化。 术语 术语 描述 layer 一个层是一个 .zip 文件（不适用于镜像），使用层技术，可以使部署包尽可能地小，因为部署包依赖的库可以通过一个新的层提供 runtime 在 Lambda 运行环境中工作的一个编程语言相关的环境，负责转发调用事件，上下文信息，以及最终的响应 handler 函数代码中由 Lambda 调用并开始执行的方法 code signing 代码签名 execution role 授权 Lambda 函数去访问 AWS 服务和资源的 IAM 角色（ AWS Identity and Access Management (IAM) role ） unpublished version 未发布的版本，当函数更新时，会发生相应的变化 published versions 已发布的版本，是函数的一个快照，不会发生变化 aliases 函数别名，可以设置使之对应于一个版本 ARN Amazon Resource Name ， 例如，execution role ARN 的样式为 arn:aws:iam::your-account-id:role/execution_role 常用命令 AWS CLI 卸载 AWS CLI version 1

AWS

简介 Amazon Web Services(AWS) 是一个云平台，拥有全球化的云基础设施，目前在全球24个地理区域运营着77个可用区。 AWS提供了大量基于云的全球性服务，其中包括： 计算 存储 数据库 分析 联网 移动产品 开发人员工具 管理工具 物联网 安全性 企业应用程序 常用命令 安装 awscli sudo apt install awscli 查看 aws 版本 aws --version 配置 awscli aws configure
